# Admin Overlay
Reportok, Duty, Offduty idő.
C# - WPF

Újra kell indítani (Overlay bezárása gomb és, ha elkezdett a szeró betölteni akkor Overlay bekapcsolása gomb), ha éjfél előtt fennvoltál, éjfél után lelépsz és újra fellépsz. (új log miatt)

Akkor frissítődik az overlay, ha történik valami, íródik új log sor + 0-10 másodperc. (pl. OOC chatre küldessz egy pontot)

Időtartamot se lehet még beállítani, szóval azokat veszi figyelembe, ami a logs mappában benne vannak console-20xx-xx-xx (dátum) fájlok. Ha, konkrét időtartamot szeretnél, akkor addig vedd ki a többi fájlt a mappából.
